# Specification

## Summary
**Goal:** Create a 2D game where players tap colored tiles in sequence, with score tracking and a leaderboard.

**Planned changes:**
- Build game interface with colored tiles that appear for players to tap
- Implement game logic to validate taps against correct color patterns
- Add score counter that updates on successful taps
- Create backend storage for high scores
- Display leaderboard showing top scores
- Apply vibrant, playful visual design with bold colors, rounded shapes, and smooth animations

**User-visible outcome:** Players can tap colored tiles in sequence, see their score update in real-time, and compete on a leaderboard showing top scores. The game features an energetic, colorful design with smooth animations.
